사전 준비: Google Cloud Platform 설정
가장 먼저 구글 API 콘솔에서 OAuth 2.0 클라이언트 ID를 발급받아야 합니다.

Google Cloud Console에 접속하여 새 프로젝트를 생성합니다.
https://console.cloud.google.com/welcome/new?authuser=5&pli=1
API 및 서비스 > 사용자 인증 정보 메뉴로 이동합니다.
사용자 인증 정보 만들기 > OAuth 클라이언트 ID를 선택합니다.
애플리케이션 유형을 웹 애플리케이션으로 선택합니다.
승인된 리디렉션 URI에 http://localhost:8080/user/googleLogin을 정확하게 추가하고 저장합니다.
생성된 클라이언트 ID와 클라이언트 보안 비밀을 복사해두세요. 이 값들은 잠시 후 Java 코드에서 사용됩니다.

📥 필요한 라이브러리 (JAR) 다운로드
Maven을 사용하지 않으므로, 아래 라이브러리(JAR 파일)를 다운로드하여 프로젝트의 WEB-INF/lib 폴더에 추가해야 합니다. 이미 Spring 및 MyBatis 관련 라이브러리가 있다는 가정하에, JSON 처리를 위한 라이브러리만 확인하면 됩니다.

요청하신 코드에 com.fasterxml.jackson.databind.ObjectMapper가 이미 사용되고 있으므로, 관련 라이브러리가 이미 포함되어 있을 가능성이 높습니다. 없다면 아래 파일들을 추가해 주세요.

Jackson Databind: JSON 데이터를 Java 객체로 변환합니다.
jackson-databind-2.13.3.jar (또는 최신 안정 버전)
jackson-annotations-2.13.3.jar
jackson-core-2.13.3.jar
